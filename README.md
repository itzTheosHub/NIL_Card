## NIL Card

## Overview
NIL Card is a platform for college athletes to create clean, shareable profile cards with social metrics and contact info — making it easy to pitch local businesses and brands for NIL partnerships.

## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Database:** Supabase (Postgres)
- **Hosting:** Vercel
- **Language:** TypeScript

## Project Principles
- Ship features in phases; finish each phase before starting the next.
- Focus on clean UI, clean data model, and reliable CRUD flows.
